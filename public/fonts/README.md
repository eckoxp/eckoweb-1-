# GE SS Two webfont files

The Arabic version of the website is configured to use `GE SS Two`.

GE SS Two is a licensed commercial font. To make it load for every visitor, place your licensed webfont files in this folder and keep/update these filenames in `src/index.css`:

- GESSTwo-Light.woff2
- GESSTwo-Regular.woff2
- GESSTwo-Medium.woff2
- GESSTwo-Bold.woff2

If the font is already installed on the visitor/device it will be used through the configured font-family. For public web usage, upload the licensed WOFF2 files here.
